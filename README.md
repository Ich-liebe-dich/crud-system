# crud-system

这是一个功能简单，基于 Spring + SpringMVC + MyBatis 开发的一个crud系统

## 开发环境

  - IDE：sts(安装有sts插件的Eclipse)
  - 项目：普通的 Dynamic Web Project
  - 数据库：MySQL 8.0.15
  - JDK：1.8

## 初始化项目
  - 下载后解压压缩包
  - 导入到Eclipse中
  - 运行src目录下的sql文件
  - 修改src目录下的db.properties里面的属性配置
  - 运行
  
## 功能
  - 登录（按照不同角色显示不同页面）
  - 对角色的增删改查，提供分页显示和按照不同条件查询的功能
  - 导出Excel文件（做的比较粗糙和繁琐）
  - 拦截器（对一些非法请求拦截，做的也十分简单）
